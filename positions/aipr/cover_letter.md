# Сопроводительное письмо — АИПР

**Канал:** отклик на hh.ru  
**Вакансия:** AI Инженер / Интегратор ИИ  
**Компания:** АИПР (Агентство исследования и прогнозирования рынков)

---

Меня заинтересовала вакансия AI Инженера: задачи конкретные — AI-контент, автоматизация аналитических процессов, интеграция инструментов ИИ в работу исследовательской компании. Именно такого рода практику я и ищу в формате частичной занятости.

Несколько слов о релевантном опыте.

По AI-контенту: в рамках продвижения платформы ВИЗАРД я веду VK-сообщество с AI-видео — генерация по промптам, трансформация изображений, анимация из статичных фото. Один из личных кейсов — из двух обычных фотографий собрал анимацию превращения себя в белого медведя, которая набрала 4 600 просмотров органически, без платного продвижения. Понимаю полный пайплайн: выбор инструмента, промпт-инжиниринг, постобработка, публикация.

По автоматизации аналитики: в проектах Роскадастр и Россельхознадзора я выстраивал аналитические пайплайны с нуля — от источника данных до структурированного результата для заказчика. Умею работать с неструктурированными данными, внешними API и собирать из этого воспроизводимый процесс.

Посмотрев на деятельность АИПР — исследования рынков РФ/СНГ/ЕАЭС, конкурентный анализ, импорт/экспорт, отчёты на 60–150 слайдов — вижу несколько точек, где AI даст конкретный эффект:

- **Инфографика и визуализация из готовых отчётов.** Автогенерация резюме исследования в формате карточек для соцсетей, "тизеров" рынков (один инсайт — одна карточка), аннотаций и highlights по ТЗ заказчика. Их ключевой продукт — PowerPoint-отчёт на 60–150 слайдов — идеальное сырьё для такого пайплайна.
- **Контент-серии "Рынок недели" / "Инсайт дня".** AI берёт срез из готового исследования → генерирует пост с инфографикой или короткое видео с озвучкой. Форматы: карточки ВКонтакте, TTS + анимация данных, Telegram. Одновременно работает как лидогенерация — демонстрирует экспертизу потенциальным клиентам.
- **AI-ассистент для квалификации клиентов.** Между кнопкой "перезвоните мне" и звонком менеджера — чат-бот, который задаёт 3–4 уточняющих вопроса (отрасль, объём рынка, сроки, бюджет), генерирует предварительную оценку стоимости и передаёт менеджеру уже квалифицированный лид с контекстом. Инструмент: n8n + Telegram-бот или встройка на сайт.

По стеку: Python, LLM-пайплайны, Agentic RAG, n8n (базовый уровень), Cursor, Windsurf. За плечами — 9 лет в AI/ML и 20 лет в ИТ в целом. Не вайб-кодер: понимаю архитектуру, умею оценить, где AI действительно помогает, а где избыточен.

Работу рассматриваю не как тренажёр, а как полноценный проект в интересной предметной области. Маркетинговые исследования — новый для меня контекст, и именно это делает задачу содержательной.
